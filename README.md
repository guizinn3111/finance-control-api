# Finance Control API

A professional REST API for personal financial management built with Node.js, Express, PostgreSQL, JWT Authentication, and Swagger Documentation.

---

# Overview

Finance Control API was developed to solve a common problem: many people do not know where their money goes throughout the month.

The project follows REST API best practices, secure authentication with JWT, PostgreSQL relational database modeling, and complete API documentation with Swagger.

---

# Features

## Authentication

* User registration
* User login
* Password encryption using bcrypt
* JWT authentication
* Protected routes

## Categories

* Create categories
* List categories
* Delete categories

Examples:

* Food
* Transportation
* Entertainment
* Health
* Education

## Transactions

* Create income transactions
* Create expense transactions
* Update transactions
* Delete transactions
* List all transactions

Examples:

* Salary
* Freelance payment
* Grocery shopping
* Fuel
* Gym membership

## Financial Goals

* Create goals
* Track progress
* Update goals
* Delete goals

Examples:

* Buy a new laptop
* Emergency fund
* Travel savings

## Reports

### Monthly Report

Provides:

* Total income
* Total expenses
* Monthly balance

### Expense Report by Category

Displays spending grouped by categories.

### Spending Alerts

Warns users when spending exceeds a predefined limit.
